### 🌡️小说解析SDK

对小说解析下载功能的封装。

> 欢迎关注公众号：【书虫无书荒】

### 主要功能

- 小说零规则目录解析及正文解析
- 书源模式（css选择器、xpath、json-path、regex）
- HTTP代理支持
- SPI自定义HTTP客户端（动 / 静态）
- 动态脚本支持
- 小说下载基础能力
